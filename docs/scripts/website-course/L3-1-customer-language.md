## שני שמות לאותו מקצוע

- חלק מהלקוחות מחפשים "שרברב", חלק מחפשים "אינסטלטור"
- שתי המילים פעילות בגוגל, ממש עכשיו, לאותו מקצוע בדיוק
- אבי צריך את שתיהן בתוכן שלו. לא רק אחת

## עכשיו תורך: תכתוב שתי רשימות

1. תכתוב 3 דרכים שבהן היית מתאר את העסק שלך לקולגה
2. תכתוב 3 דרכים שבהן לקוח בבהלה היה מקליד את זה בגוגל
3. תשים את שתי הרשימות זו לצד זו, ותשווה
4. רוצה לוודא שלא ניחשת? תתחיל להקליד בגוגל, ותראה מה הוא משלים לבד
